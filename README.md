# PRISM Models Registry

Official model repository for PRISM Visual Pipeline application.

## Models Available
- **Panel Detection**: Comic panel layout detection
- **LLaVA 1.5 7B**: Visual analysis and descriptions  
- **LLaVA Vision**: Vision processing component
- **SAM 2**: Image segmentation (Small/Base+/Large variants)

## Usage
Models are automatically downloaded by the PRISM application based on your hardware configuration.
